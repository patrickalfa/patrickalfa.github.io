Drag-Draft

A Tactics/Card game prototype for mobile devices.

  • In this game CARDS ARE ACTIONS. You can deploy them over your heroes to perform actions on them;
  • To deploy a card you have to spend the number of Action Points shown on the card. You start each round with 3 AP;
  • Some cards can only be deployed on a specific hero class. You can see that by the color of the cards:
    • GRAY cards can be deployed on all heroes;
    • COLORED cards can only be deployed on the heroes of the matching color;
    • BLACK cards can't be deployed;
  • Hold down a card for more details on it.
